## AI内容编辑器-智能格式处理工具

一个功能强大的AI内容编辑器，专为处理AI生成的Markdown内容而设计。支持内容编辑、格式转换和多格式导出，特别适合处理含有特殊标记和格式的AI生成内容。

### 主要功能

#### 编辑与预览

双栏布局：左侧编辑区，右侧实时预览区
智能粘贴：支持粘贴导入AI生成的带Markdown格式的内容
实时预览：编辑内容实时转换为格式化文本
一键复制：快速复制格式化后的内容（含序号）

#### 多格式导出

Word导出：完整内容导出为.docx格式文档
PDF导出：通过打印功能导出为PDF文件
Excel导出：表格内容导出为.xlsx格式表格
格式保留：完美保留原始格式和样式

#### 智能处理

自动序号：智能识别并添加列表序号（1. 2. 3. 和 ① ② ③）
格式清理：自动处理AI生成内容中的特殊标记
表格识别：智能识别和格式化表格内容
代码高亮：支持代码块语法高亮显示

### 使用方法

1. 内容输入：在左侧编辑区粘贴或输入AI生成的Markdown内容
2. 实时预览：右侧即时显示格式化的预览效果
3. 内容复制：点击"复制内容"将格式化内容复制到剪贴板
4. 文档导出：
   点击"导出Word"导出完整文档为Word格式
   点击"导出PDF"通过打印界面保存为PDF
   点击"导出Excel"导出表格内容为Excel格式

### 技术特色

纯前端架构：HTML/CSS/JavaScript实现，无需服务器支持
离线运行：完全本地处理，保护数据隐私
轻量高效：快速加载，响应式交互体验
跨平台兼容：支持所有现代浏览器

### 核心优势

AI内容优化：专门针对AI生成内容进行格式优化
符号处理：智能处理段首"-"、"·"等特殊符号
嵌套格式：完美支持嵌套列表和复杂格式
表格对齐：自动优化表格对齐和格式
隐私保护：纯前端处理，数据不上传服务器
